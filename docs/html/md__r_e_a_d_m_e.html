Lanciare il programma

Per lanciare il programma, basta eseguire il comando "`docker compose up`" nella cartella root del progetto. Questo comando lancia i container dove sono presenti i servizi necessari per il funzionamento del programma:

  • redis
  • postgres
  • server
  • main

Visualizzare l'area in tempo reale

Per visualizzare l'area in tempo reale, basta eseguire lo script python nel file area GUI.py presente nella root.
